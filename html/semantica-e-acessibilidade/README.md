# HTML Semântico e Acessibilidade

Este projeto tem como objetivo a **revisão e o aprofundamento dos conceitos de HTML semântico e acessibilidade**, abordando boas práticas recomendadas para o desenvolvimento de interfaces web mais **estruturadas, acessíveis e alinhadas aos padrões da Web**.

Ao longo do projeto, são aplicadas **tags semânticas do HTML5**, com foco em:

- Melhor organização e legibilidade do código  
- Maior acessibilidade para usuários de tecnologias assistivas  
- Melhor interpretação do conteúdo por mecanismos de busca (SEO)

Além disso, o projeto explora princípios fundamentais de **acessibilidade web**, como:

- Uso correto da hierarquia de títulos (`h1` a `h6`)  
- Aplicação adequada do atributo `alt` em imagens  
- Utilização de elementos semânticos para navegação e estrutura  
- Inclusão de conteúdos acessíveis a leitores de tela (`sr-only`)

Este repositório serve como **material de estudo** e também como **referência prática** para o desenvolvimento de páginas web mais inclusivas, organizadas e semanticamente corretas.
